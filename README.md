# Ferone Tours & Travels

Premium Next.js 15 website for **FERONE TOURS & TRAVELS** — Explore Beyond Borders.

## Stack

- Next.js 15 App Router
- React + TypeScript
- Tailwind CSS
- Framer Motion
- SEO metadata and JSON-LD structured data
- Responsive premium travel UI

## Pages

- Home
- About Us
- Tours & Safaris
- Destinations
- Visa Services
- Flight Booking
- Gallery
- Blog
- Contact

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Deploy

Import this repository into Vercel, then set:

```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.com
NEXT_PUBLIC_WHATSAPP_NUMBER=254712345678
```

## Image replacement

The project is arranged so public images live under `/public/images`. Replace files with real optimized travel photos using the same filenames or update the paths in `lib/data.ts`.
